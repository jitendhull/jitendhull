### 🏠 My Digital Playground (HomeLab)
I run a personal HomeLab where I self-host various services for both learning and daily utility. This setup allows me to experiment with enterprise-grade networking and virtualization in a controlled environment.

- **Infrastructure:** Managed via **Docker & Docker Compose** for high portability.
- **Server Management:** Utilizing **Pterodactyl Panel** for game server orchestration and resource monitoring.
- **Routing:** **Nginx** configured as a reverse proxy for secure, external access to local services.
- **Automation:** Leveraging **Rclone** for off-site backups and cloud synchronization.
- **OS:** Primarily powered by **Linux (Arch/CachyOS)** for performance and deep system control.
